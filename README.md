# FUELMETER ⛽
**Smart Vehicle Fuel Management App**

## 📖 Introduction
FuelMeter is a cross-platform Flutter application designed to monitor multi-vehicle fuel consumption, forecast fuel economy and range, and predict optimal refill times. Integrated with a Flask backend, custom Machine Learning models, and Firebase, it enables users to seamlessly log data, manage multiple vehicles, and receive real-time predictive insights through an intuitive user interface.

## ✨ Key Features
* **Multi-Vehicle Management:** Access the Vehicle Settings screen via the side navigation panel to seamlessly add, edit, delete, and view details for multiple vehicles.
* **Comprehensive Refill Tracking:** Manage fuel refill records directly from the home screen or side navigation. Refills are automatically grouped and sorted by date for easy tracking.
* **AI-Powered Consumption Prediction:** Enter vehicle details to generate predictive insights on exactly when and where you should refuel next.
* **Real-Time Dashboard Updates:** The home screen automatically synchronizes with the ML models to display predicted fuel usage and your expected refueling date.
